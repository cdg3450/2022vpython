## [프로그램 개요]
내가 만들고 싶은 프로그램은 자동 주식 투자와 예측이 가능한 투자 프로그램을 만드는 것이다. 일반적으로 주식 투자를 하면 매시간 집중하고 있어야 해서 다른 일을 하기 어려워지지만 이 프로그램을 만들어 편리하게 투자를 해보고 싶다.
